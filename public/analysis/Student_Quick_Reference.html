💡 Important Formulas & Shortcuts

Number Conversions:

Binary → Decimal: Multiply each bit by 2^position
Decimal → Binary: Keep dividing by 2, take remainders
Binary → Octal: Group into 3 bits from right
Binary → Hex: Group into 4 bits from right

Spreadsheet $ Reference:

A1 = Both column & row change when copied
$A1 = Column fixed, row changes
A$1 = Column changes, row fixed
$A$1 = Both fixed (absolute reference)

Logic Gates:

AND: Output 1 only if ALL inputs are 1
OR: Output 1 if ANY input is 1
NOT: Reverses the input
NAND: NOT(AND) - Opposite of AND
NOR: NOT(OR) - Opposite of OR

HTML Common Tags:

<ol> ordered list • <ul> unordered list • <li> list item
<table> • <tr> row • <th> header • <td> data
<a href=""> hyperlink • <img src=""> image
<h1> to <h6> headings • <p> paragraph

⚠️ Common Mistakes to AVOID

  • ❌ Don't forget $ symbol in absolute cell references
  • ❌ Don't confuse Primary Key with Foreign Key
  • ❌ Don't forget closing HTML tags (</p>, </table>)
  • ❌ Don't mix up Direct vs Parallel deployment
  • ❌ Don't write Pascal code with case errors (begin, end, var)
  • ❌ Don't forget <tr> before each table row
  • ❌ Don't ignore the difference between RAM & ROM

⏰ Last-Minute Exam Tips
  1. Read all 7 structured questions first, then choose your best 4
  2. Q1 is compulsory - manage time well (20 marks)
  3. Show working for number conversions - partial marks!
  4. Draw clear truth tables with column headers
  5. Use ruler for diagrams, flowcharts, and logic circuits
  6. Label everything in flowcharts (Start, Stop, Input, Output)
  7. Don't leave blanks in MCQ - educated guesses are better
  8. Re-check ASCII conversions - common careless mistakes
